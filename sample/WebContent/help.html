Welcome

Recommendations is an online application showcasing the restricted scope of the proposed design project under Glance brief in HCI.
Here a user can look for a product information, customer reviews, video previews and ratings from an online community by searching on the top search bar.
Keeping the scope of the project restricted as per the deadline for conducting experiments, search functionality currently supports few products only which are listed below. In order to mark your feedback on the work, please use the following product search only.

Category Products
Books Freakonomics
The Art of Computer Programming
Mobile & Accessories Samsung Galaxy S3
Samsung Galaxy Tab
Note: If you select "All Categories" and search for any of the product given above, it will show the results. But if you select a category as given above and then selected a product which does not belong to that category, it will show no results.
